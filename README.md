# Subdomain-crawler
子域名爬虫脚本

一个用python编写的简单的爬取各在线网站子域名的爬虫

##### 包括以下网站:

https://spyse.com/

https://fofa.so/

https://www.bing.com/

https://www.baidu.com/

https://hackertarget.com/

https://crt.sh/

https://www.threatcrowd.org

https://chaziyu.com/

##### 使用方法

1.安装相关库
2.填入fofa :_fofapro_ars_session,spyse api.
